面试题10_斐波那契数列

本页题解有四个部分，包括斐波那契数列及其扩展——青蛙跳台阶、变态跳台阶以及矩形覆盖问题。

题目描述

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。n<=39

斐波那契数列

解法一

最简单的解法就是递归了，相信大家都会写。

public int Fibonacci1(int n) {
    if (n == 0 || n == 1)
        return n;
    return Fibonacci(n-1) + Fibonacci(n-2);
}

解法二

动态规划 dp[n] = dp[n-1] + dp[n-2]

public int Fibonacci(int n) {
    if (n == 0 || n == 1)
        return n;
    int[] dp = new int[n+1];
    dp[0] = 0; dp[1] = 1;
    for (int i = 2; i <= n; i++)
        dp[i] = dp[i-1] + dp[i-2];
    return dp[n];
}

青蛙跳台阶

解法一

动态规划 dp[i] = dp[i-1] + dp[i-2]

public int JumpFloor(int target) {
    if (target < 3)
        return target;
    int[] dp = new int[target+1];
    dp[1] = 1;
    dp[2] = 2;

    // dp[i] = dp[i-1] + dp[i-2]
    for (int i = 3; i <= target; i++)
        dp[i] = dp[i-1] + dp[i-2];

    return dp[target];
}

变态跳台阶

解法一

动态规划 dp[i] = dp[0] + dp[1] + ... + dp[i-1]

public int JumpFloorII(int target) {
    int[] dp = new int[target+1];
    dp[0] = 1;

    // dp[i] = dp[0] + dp[1] + ... + dp[i-1]
    for (int i = 1; i <= target; i++)
        for (int j = 0; j < i; j++)
            dp[i] += dp[j];

    return dp[target];
}

解法二

由于最后一个台阶必须存在，前n-1个台阶可以任意选择是否存在，因此前n-1个台阶有2*(n-1)个选择。

public int JumpFloorII(int target) {
    if (target == 1)
        return 1;
    return JumpFloorII(target-1) * 2;
}

矩形覆盖

解法一

递归

public int RectCover(int target) {
    if (target < 1)
        return 0;
    if (target == 1)
        return 1;
    if (target == 2)
        return 2;
    return RectCover(target-1) + RectCover(target-2);
}

解法二

动态规划 dp[i] = dp[i-1] + dp[i-2]

public int RectCover(int target) {
    if (target < 3)
        return target;
    int[] dp = new int[target+1];
    dp[1] = 1;
    dp[2] = 2;
    for (int i = 3; i <= target; i++)
        dp[i] = dp[i-1] + dp[i-2];
    return dp[target];
}
